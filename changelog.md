# Resources Changelog

## 0.2.0

### New/Changed Features

* Add support for the QEMU hypervisor
* Add support for Linux as the host
* Add support for the following target architectures:
    * aarch64
    * alpha
    * arm
    * hppa
    * i386
    * m68k
    * mips
    * mips64
    * mips64el
    * mipsel
    * ppc
    * ppc64
    * riscv32
    * riscv64
    * s390x
    * sparc
    * sparc64
    * x86_64
